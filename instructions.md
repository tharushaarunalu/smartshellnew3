# Smart Shelf | Supermarket Item Locator (Premium Update)

This is a modern, high-end web application designed to help customers find items in a supermarket using an interactive SVG map and smart navigation.

## 🚀 Premium Features
- **Glassmorphism UI**: A sleek, modern design with blurred surfaces and vibrant gradients.
- **Interactive SVG Map**: Dynamic rack generation for Ground, 1st, and 2nd floors.
- **Smart Pathfinding**: Visualizes the path from the 'Entrance' to the item's rack using an animated dashed line.
- **Auto-Floor Switching**: Searches automatically switch to the correct floor view.
- **Premium Icons**: Uses Lucide icons for a professional interface.
- **Admin Dashboard**: Manage items, floor rack counts, and upload product images.
